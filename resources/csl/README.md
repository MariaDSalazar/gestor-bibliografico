# Estilos CSL

- `ieee.csl` — IEEE. Lo usa `CiteprocCitationFormatter` (citeproc-py) para generar
  las referencias en formato IEEE.

Fuente: https://github.com/citation-style-language/styles (CC BY-SA 3.0 — conservar atribución).

> APA 7 y BibTeX se generan con un formateador propio (en español), no con CSL,
> porque citeproc-py 0.6.0 no procesa el `apa.csl` moderno. El locale `es-ES` lo
> aporta citeproc-py de su propio paquete.
